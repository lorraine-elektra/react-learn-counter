# react-counter
This is one of my first React projects, built as part of my learning journey as a beginner developer. It is a simple counter exercise that helped me get a real feel for how React works. Everything here was built step by step through problem solving, not by copying solutions.

## What I Built
A button that counts up every time you click it. There are two counters sharing the same count and a Reset button that sets everything back to zero.

## What I Learned
Building this gave me real hands on experience with the core building blocks of React. Things like components, state, props and making pages interactive with events started to actually click for me through this exercise.

This is a learning project and a starting point, not a finished product. More projects to come as I keep building.
